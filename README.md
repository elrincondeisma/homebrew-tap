# homebrew-tap

Tap de Homebrew de [elrincondeisma](https://github.com/elrincondeisma).

```bash
brew tap elrincondeisma/tap
brew trust elrincondeisma/tap   # Homebrew moderno lo exige para taps de terceros
brew install --cask ecs-monitor
```

## Casks

- **ecs-monitor** — [ECS Monitor](https://github.com/elrincondeisma/desktop-monitor-ecs): monitor de AWS ECS en la barra de menú
- **keypet** — [KeyPet](https://github.com/elrincondeisma/keypet): mascota pixel que reacciona a tu teclado
